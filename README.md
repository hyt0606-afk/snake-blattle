# 🐍 Snake Blattle - 多人貪吃蛇遊戲

一個簡單有趣的網頁版貪吃蛇遊戲，支持本地排行榜和多玩家競技。

## 功能特性

✨ **遊戲玩法**
- 使用方向鍵控制蛇的移動（↑ ↓ ← →）
- 吃掉紅色食物增加分數
- 避免撞到牆壁或自己的身體
- 遊戲結束會顯示最終分數

🏆 **排行榜系統**
- 自動保存玩家的最高分
- 本地存儲前 10 名玩家成績
- 實時更新排行榜

🎮 **多玩家支持**
- 輸入昵稱開始遊戲
- 支持多個玩家輪流挑戰
- 追蹤每位玩家的成績

## 如何遊玩

1. 打開 `index.html` 文件在瀏覽器中
2. 輸入你的昵稱
3. 點擊「開始遊戲」按鍵
4. 使用方向鍵控制蛇的移動
5. 吃掉食物獲得分數
6. 遊戲結束後查看排行榜

## 技術棧

- **HTML5** - 頁面結構
- **CSS3** - 樣式設計
- **JavaScript** - 遊戲邏輯和交互
- **Canvas API** - 遊戲繪製
- **LocalStorage** - 數據持久化

## 遊戲規則

- 🟢 綠色方塊 = 你的蛇
- 🔴 紅色方塊 = 食物
- 每吃到一個食物增加 1 分
- 蛇碰到邊界或自己會遊戲結束
- 分數會自動保存到排行榜
